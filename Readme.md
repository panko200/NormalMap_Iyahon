NormalMap_Iyahon
製作者：Panko200
配布場所：https://github.com/panko200/NormalMap_Iyahon

## 概要

ノーマルマップを読み込みできるようにした。つもり。ノーマルマップを一回も触ったことないのに作り始めたから、実際のノーマルマップがどんなのかを知らぬまま作りました。なんか違うところがあっても許してください。ついでにハイトマップも触ったことなかったのでよくわかりません。

## 使用方法

映像エフェクト「NMI ノーマルマップ生成」「NMI ノーマルマップ」「NMI 光源設定」が追加されます。

・「NMI ノーマルマップ生成」では、入力された画像のノーマルマップを作ってくれる。ただし、次に紹介する NMI ノーマルマップとエフェクト間で連携することは、私の現在の技術的に厳しかったので実装していません。

・「NMI ノーマルマップ」では、アイテムにノーマルマップをはっつけることができます。次に紹介する NMI 光源設定によって、光の位置を調整して、凸凹を見ることができます。また、NMI 光源設定を使う場合、このエフェクトと、NMI 光源設定の光源 ID は同じ値にしてください。

・「NMI 光源設定」では、光源の位置などを決めることができます。NMI ノーマルマップと NMI 光源設定の光源 ID は同じ値にしてください。また、光源設定は、各光源 ID につき、1 つまでしかおけません。なんか二つ以上置くとクッソちらつきます。

## アンインストール方法

1. YMM4 を起動して`ヘルプ(H)`>`その他`>`プラグインフォルダを開く`をクリックする。
2. YMM4 を終了する。
3. `NormalMap_Iyahon`という名前のフォルダを削除する。

## 注意点

ゆっくりムービーメーカー v4.47.0.0 以上を使用してください。

グループ制御に光源設定などをつけないでください。動作しません。複数のアイテムに NMI ノーマルマップをつけていて、それをグループ制御で管理している場合は、グループ制御外に、なんでもいいのでアイテムを配置して、不透明度を 0 にして、光源設定を適応してください。まぁつまりはヌルのように扱ってね……

3D 空間で扱う場合、向きが大切になってきます。3D 回転と左右反転を駆使してもらうといいかなと思います。

プラグイン名に Twitter 上の自分の名前(イヤホン)が入っているのは、今後ほかの人がノーマルマッププラグインを作ったとき、名前が被らないようにするためです。

実は、ハイトマップは YMM4 標準にも入っているって皆さん知っていましたか？
